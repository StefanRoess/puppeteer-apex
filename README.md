# puppeteer-apex
Application Express will be controlled via node.js API Puppeteer

### install puppeteer
<code>npm install puppeteer --save</code>

ends up in a package.json  
<code>  "dependencies": {  
    "puppeteer": "^1.0.0"  
  }  
</code>

### uninstall puppeteer
<code>npm uninstall puppeteer --save</code>
